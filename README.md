# Blog App

A simple **Blog Application** built with React that allows users to create, delete, and view blog posts.  
This project demonstrates core frontend development concepts like **component-based architecture, and dynamic UI updates**.

## Features

- Create new blog posts
- View all blog posts
- Delete blog posts
- Responsive user interface
- Dynamic rendering using React

## Tech Stack

- React.js
- JavaScript (ES6)
- HTML5
- Tailwind CSS

## Installation

1. Clone the repository

```
git clone https://github.com/SaloniKasaudhan123/Blog-App.git
```

3. Install dependencies

```
npm install
```

4. Start the development server

```
npm start
```

The app will run on:

```
http://localhost:3000
```

## Usage

1. Open the application in the browser.
2. Create a new blog post.
3. Delete existing posts.
4. All changes will update instantly in the UI.
